# 🎬 Movie Recommender System

🚀 **Live App:** https://movie-recommender-wu8bhudemct7wnacwv5bmc.streamlit.app/

A content-based movie recommendation system that suggests movies based on user selection. Built using Python, Streamlit, and Machine Learning techniques.

---

## 🚀 Features

- Recommend similar movies based on selected movie
- Displays movie posters
- Fast and interactive UI using Streamlit
- Uses cosine similarity for recommendations

---

## 🛠️ Tech Stack

- Python
- Pandas, NumPy
- Scikit-learn
- Streamlit
- Pickle
